Detailed Description

MessageService class provides pub/sub messaging infrastructure. Used for publishing notifications of occurences within the system.

Using this instead of tradditional events decouples the various components from each other to the extent that each component is only aware of the messages it is interested in an not who is publising the message at any point in time.

Note: Currently this is not designed for high-volume messages. But is fine for ad-hoc in game notifications.
